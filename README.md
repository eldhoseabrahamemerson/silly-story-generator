# 🎲 Silly Story Generator

A fun JavaScript project that generates random silly stories with customizable names and locale-based unit conversions. This project was completed as part of the **MDN Web Docs – First Steps** module and the **µLearn Foundation** web development track.

## 🚀 Live Demo

**GitHub Pages:**
https://eldhoseabrahamemerson.github.io/silly-story-generator/

## 📖 About the Project

The Silly Story Generator creates a new random story every time the **Generate Random Story** button is clicked. Users can personalize the story by entering a custom name and choose between US and UK locales for automatic temperature and weight conversions.

## ✨ Features

* 🎲 Generates a new random story on every click
* 👤 Supports custom character names
* 🌍 Switch between US and UK units
* 🌡️ Converts Fahrenheit to Celsius
* ⚖️ Converts Pounds to Stone
* 📱 Responsive and clean user interface
* 💻 Built using vanilla HTML, CSS, and JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
